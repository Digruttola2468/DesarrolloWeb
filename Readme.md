## Tipos de selectores mas usados
- DE tipo     -->    `div{}`
- De clase    -->    `.elemento{}`
- De ID       -->    `#id-del-elemento`
- De Atributo -->    `a[href=""]{}`
- Universal   -->    `*{}`
